# Jogo de Dados

![](https://imgur.com/MUVfv60.gif)
## Descrição

Este é um simples jogo de dados desenvolvido em C# para console, onde o usuário compete contra o computador em uma corrida para alcançar a linha de chegada. O jogo envolve o lançamento de dados, eventos especiais que podem ajudar ou atrapalhar o progresso de cada jogador, e a possibilidade de ganhar uma rodada extra ao tirar um "6".

O objetivo do jogo é ser o primeiro a alcançar a linha de chegada (definida como a posição 30). O jogador e o computador alternam turnos para lançar os dados e mover suas posições no tabuleiro, com eventos especiais que podem alterar o progresso de ambos.

## Funcionalidades

- **Lançamento de Dados**: O jogador e o computador lançam um dado para determinar quantas casas se movem.
- **Eventos Especiais**:
  - Avanço extra de 3 casas em posições específicas.
  - Recuo de 2 casas em outras posições específicas.
  - Rodada extra ao tirar um "6" no dado.
- **Interatividade**: O jogo exibe o progresso dos jogadores após cada turno e permite ao usuário continuar jogando ou finalizar a partida.

## Como Jogar

1. Execute o programa.
2. O jogo começa com o turno do jogador (você).
3. O jogador lança o dado e move sua posição no tabuleiro.
4. O computador então joga e move sua posição de acordo com o valor sorteado no dado.
5. O jogo exibe os eventos especiais, como avanço ou recuo de casas, e o status do jogo.
6. O primeiro a alcançar ou ultrapassar a linha de chegada (posição 30) vence. O jogo permite jogar novamente após cada partida.
